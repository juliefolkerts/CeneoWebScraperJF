About the Ceneo Webscraper Project

Welcome to the Ceneo Webscraper project homepage! This project is designed to help users quickly and easily find reviews for products listed on Ceneo.pl, a popular Polish price comparison website.

Using my web application, you can simply enter the product code of any item you're interested in, and the system will fetch and display the reviews available on Ceneo.pl. This can help you make informed purchasing decisions by providing you with genuine user feedback.

The Ceneo Webscraper is built using the following technologies:

  • HTML - for the structure of the web pages.
  • Flask - a lightweight web framework for Python, which powers the backend.
  • Bootstrap - for a responsive and modern CSS design, ensuring the site looks great on all devices.

I hope you find this tool helpful!
